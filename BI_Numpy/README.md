## 🚀 Sobre o Projeto
Este repositório foi criado para centralizar as atividades, laboratórios e projetos desenvolvidos na disciplina de **Business Intelligence** da Faculdade SENAI Fatesg. O foco principal é a transformação de dados brutos em insights estratégicos, utilizando técnicas modernas de análise e automação.

## 🛠️ Tecnologias e Conceitos
O curso abrange a integração de diferentes pilares tecnológicos:

* **Python:** Utilizado para processamento de dados (ETL), limpeza com *Pandas* e automação de scripts.
* **Business Intelligence (BI):** Modelagem de dados (Star Schema/Snowflake), criação de Dashboards e indicadores de performance (KPIs).
* **Inteligência Artificial (IA):** Aplicação de modelos preditivos e análise de dados assistida para suporte à decisão.
* **SQL:** Consultas estruturadas para extração de dados de bancos OLTP e carregamento em Data Warehouses.

## 📂 Estrutura do Repositório
* `/atividades`: Exercícios práticos realizados em sala.
* `/datasets`: Conjuntos de dados utilizados nos estudos de caso.
* `/scripts`: Códigos Python para tratamento de dados.
* `/dashboards`: Arquivos de visualização (Power BI/Tableau).
